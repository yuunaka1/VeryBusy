# Security Policy

## Supported Versions

Since `VeryBusy` is a visual simulator intended for demonstration purposes only, we do not provide backported security patches for older versions. Please aim to use the latest version available on the main branch.

| Version | Supported          |
| ------- | ------------------ |
| Latest  | :white_check_mark: |

## Reporting a Vulnerability

`VeryBusy` operates entirely locally and does not open network ports, parse external untrusted inputs, or communicate over the network. It is a strictly cosmetic terminal simulation tool.

However, if you discover a security vulnerability (e.g., in an upstream dependency or a local sandbox escape flaw), please do NOT open a public issue.

Instead, please privately report it via the standard GitHub Private Vulnerability Reporting mechanism for this repository. We will acknowledge receipt and work on a fix as quickly as possible.
